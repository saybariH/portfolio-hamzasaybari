Real-time person detection and face identification

In this project, I collected facial data of six actors from various scenes in a TV show. Using YOLO (You Only Look Once) to detect the person, I then extracted the face from the detected region. Finally, I identified the actors using a Convolutional Neural Network (CNN) model for face recognition.

Facial-Based Age and Gender Detection

This project involved collecting and cleaning a dataset of over 25,000 facial images. We conducted experiments with different models to achieve optimal results for facial analysis tasks such as age and gender detection. Finally, we successfully implemented the chosen model into a web application for practical usage.

Healthcare stroke causes analysis

In this project, I used association rule mining to analyze healthcare data and identify patterns linked to stroke risk factors. By applying the Apriori algorithm, I uncovered relationships between health conditions and lifestyle choices.

Image Processing using Computer Vision Techniques

In this project, I applied various computer vision techniques for image processing tasks. By leveraging algorithms such as Gaussian filters for smoothing and median filters for noise reduction, I enhanced image quality and clarity. These techniques were essential for improving the accuracy of subsequent image analysis steps.

Heart Disease analysis

In this project, I analyzed a dataset related to heart disease using Apache Spark for efficient data processing and analysis. The project involved cleaning and preprocessing the data, followed by exploratory data analysis to uncover key trends and insights. I utilized Spark's machine learning library (MLlib) to build and train a predictive model that identifies the likelihood of heart disease based on various health indicators.

Human Ressources Management App

In this project, I developed a generic Human Resources Management application using Java and Spring Boot. The app allows for managing employee data, attendance and leave requests. Spring Boot was used to build a scalable, RESTful backend, enabling smooth handling of HR operations and providing secure user authentication and role-based access.

Tourists Guide web site
